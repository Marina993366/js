# JAVASCRIPT
# JavaScript Learning Path: De Fundamentos a Lógica Avanzada 🚀

Este repositorio contiene mi progresión de aprendizaje en JavaScript, desde los conceptos más básicos de sintaxis hasta la manipulación avanzada de datos y lógica de programación.

El objetivo de este proyecto es documentar mi evolución técnica, resolviendo desafíos de creciente complejidad que simulan situaciones reales y juegos de lógica.

## 📂 Estructura del Proyecto

Los ejercicios están organizados de forma secuencial para reflejar el orden de aprendizaje:

1.  **`1-Fundamentos-variables-y-TIPOS.js`**: Introducción a la interacción con el usuario, declaración de variables (`let`, `const`) y operaciones aritméticas básicas.
2.  **`2-operadores-logicos-condicionales.js`**: Implementación de lógica booleana (`&&`, `||`) y estructuras de decisión (`if/else`, `switch`) para validación de datos.
3.  **`3-estructuras-de-control-simuladores.js`**: Creación de simuladores (como un control remoto de TV) utilizando estados y validación de arrays.
4.  **`4-bucles-iteracion-logica-de-juegos.js`**: Uso de ciclos (`for`, `while`) y aleatoriedad (`Math.random`) para crear minijuegos como "Piedra, Papel o Tijera" o "Adivina el Número".
5.  **`5-Metodos-de-arrays-y-objetos.js`**: Programación funcional aplicada. Uso de métodos de orden superior como `.map()`, `.filter()` y `.reduce()` para agrupar y procesar colecciones de objetos complejos.

## 🛠️ Tecnologías y Conceptos Clave

- **Sintaxis Moderna**: ES6+.
- **Estructuras de Control**: Bucles, condicionales y manejo de errores.
- **Manipulación de Arrays**: Métodos iterativos y transformación de datos.
- **Lógica de Algoritmos**: Búsqueda de máximos/mínimos y contadores de rondas.
